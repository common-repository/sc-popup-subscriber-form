jQuery Popup Feedburner Subscriber Form.

== Description ==
jQuery Popup Feedburner Subscriber Form appears on Page Load.

You can
1. Enable/disable jQuery Popup Subscriber Form from admin panel
2. You can set the feedburner feed id from admin panel
3. You can change the Heading and Detail from Popup form from admin panel

For Support visit: http://sharp-coders.com/plugins/wp-plugins/sc-popup-subscriber-form-wordpress-plugin

== Installation ==

= Step 1 =

1. Download zip file and Extract it 
2. Upload sc-popup-subscriber-form folder to your /wp-content/plugins/ folder.
2. Go to the 'Plugins' page in the menu and activate the plugin.

= Step 2 =

1. Go to Wodpress Admin Panel > Setting > SC Popup Subscriber
2. Update Feed Address, Heading and Detail.
